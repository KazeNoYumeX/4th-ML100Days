# 4th-ML100Days

機器學習百日馬拉松是一個結合自學、專家協助及社群共學的 AI 自學挑戰活動，
目的是希望學員從參與的那天起，每天至少花一小時，有系統的學習資料科學與機器學習的相關知識點，
並進行 AI 程式練習，連續堅持 100 天，進而更好理解掌握資料科學與 AI 程式實作的能力。
